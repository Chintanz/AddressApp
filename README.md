# AddressApp (Secure Profile Sync)

## Run the App
1. Install dependencies:
   ```bash
   cd /workspaces/AddressApp
   npm install
   ```
2. Start server:
   ```bash
   npm start
   ```
3. Open browser at `http://localhost:3000`

## API Endpoints
- GET `/api/profile`
- PUT `/api/profile`
- GET `/api/entities`
- POST `/api/entities`
- POST `/api/sync`
- GET `/api/events`
